Q1
What are the possible ways to compare two variables in JavaScript?
- Equality: ==
- Inequality: !=
- Identity / strict equality: ===
- Non-identity / strict inequality: !==
- Greater than: >
- Greater than or equal: >=
- Less than: <
- Less than or equal: <=

Can variables with the same value still not be equal?
Yes, two variables can hold the same values but have different types e.g. 1 === '1' -> false
Furthermore, two variables can hold the the same value and be of the same type (Object in the example) but reference different objects e.g. new String('foo') === new String('foo') -> false
